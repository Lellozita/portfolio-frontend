# Portfolio Frontend
Projeto básico de Frontend criado com HTML, CSS e JavaScript.

# Objetivo: 
praticar conceitos fundamentais de desenvolvimento web, criando uma landing page simples com seções "Sobre Mim", "Projetos" e "Contacto".

# Funcionalidades:
- Layout simples e responsivo
- Botão para mostrar/esconder o e-mail de contacto
- Estrutura organizada em arquivos separados (HTML, CSS e JS)

# Tecnologias utilizadas: 
HTML, CSS, JavaScript
